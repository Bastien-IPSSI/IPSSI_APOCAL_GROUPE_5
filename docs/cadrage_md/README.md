# 📂 Cadrage Markdown (Source of Truth)

Ce dossier contient l'intégralité des artefacts de cadrage du projet EduTutor IA, convertis au format Markdown pour permettre le versionnage et la collaboration agile.

**⚠️ Note :** Ces fichiers remplacent les documents binaires (`.docx`, `.xlsx`) situés dans `docs/cadrage/`.

## 📑 Sommaire
- [Vision Board](vision_board.md) : Objectifs et valeur ajoutée du produit.
- [Personas](personas.md) : Profils détaillés de l'Étudiant et de l'Enseignante (Mme Lefèvre).
- [Customer Journeys](customer_journeys.md) : Parcours utilisateurs et émotions.
- [Product Backlog](product_backlog.md) : User Stories INVEST et priorisation MoSCoW.
- [Story Map](story_map.md) : Structure du produit et planification des releases.
- [Note de Décision J1](decision_note_j1.md) : Justification stratégique pour l'ajout de la cible Enseignante.

## 🛠️ Comment mettre à jour ?
Pour modifier un artefact, éditez simplement le fichier `.md` correspondant et poussez vos modifications sur GitHub.
