# Product Backlog — Ajouts perturbation Mme Lefèvre

J'ai ajouté 3 user stories pour la persona **Mme Lefèvre** au format CSV importable. Fichier créé : `docs/cadrage/product-backlog.additions.csv`.

## Contenu (résumé)

- 101 — Dashboard synthétique (SHOULD, 5 pts) : vue globale de la classe, indicateur risque, tri/filtre.
- 102 — Filtrer & marquer élèves (SHOULD, 3 pts) : filtres et marquage 'à suivre'.
- 103 — Envoyer message template (COULD, 3 pts) : envoi de templates aux élèves / groupes.

## Comment intégrer dans `product-backlog.xlsx` (Excel / Numbers)

1. Ouvrez `docs/cadrage/product-backlog.xlsx` dans Excel ou Numbers.
2. Dans Excel : onglet **Données** → **À partir du texte/CSV** → sélectionnez `product-backlog.additions.csv`.
3. Dans la fenêtre d'import, choisissez le séparateur `,` et vérifiez l'encodage UTF-8, puis **Charger**.
4. Copiez les lignes importées et collez-les à la fin de la feuille `Product Backlog` (ou utilisez l'option d'import pour l'ajouter directement).
5. Enregistrez le fichier `.xlsx`.

## Si vous voulez que je l'insère directement dans le .xlsx

Dites 'insère' et je mettrai à jour `product-backlog.xlsx` directement et je vous décrirai les changements.
